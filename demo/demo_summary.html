PatchSteg

Covert Steganographic Communication Between Vision Models via Latent Space Perturbation

Training-free Post-hoc CDF-PatchSteg: AUC ≈ 0.5 No GPU required

What is PatchSteg?

PatchSteg embeds binary messages in images by making small, structured changes in the latent space of a pretrained VAE (Stable Diffusion VAE). Two AI agents sharing a secret seed can send hidden instructions through images that appear pixel-identical to human observers.

Key novelty: PatchSteg works on existing images — post-hoc, with no training, no fine-tuning, and no generation-time control. All prior distribution-preserving methods (Gaussian Shading, PRC, PSyDUCK) require injecting the secret during diffusion sampling.

Attack Methods (available in demo UI)

Original PatchSteg (±ε)

Shifts each carrier latent value by +ε (bit=1) or −ε (bit=0) along a shared direction vector. Fast and accurate but statistically detectable at high ε.

Bit accuracy~100% (ε=5)
PSNR~29 dB (K=20)
Detection AUC0.93
Blind decodeNo (needs cover)
CDF-PatchSteg — SOTA (undetectable)

Resamples each carrier value from the upper (bit=1) or lower (bit=0) half of N(μ,σ) via inverse Gaussian CDF. The latent distribution is preserved exactly.

Bit accuracy~88–92% (K=5–20)
PSNR~37–40 dB
Detection AUC≈ 0.50 (chance)
Blind decodeYes (no cover needed)

Running the Demo

Live Gradio UI (requires Python)

source ~/patchseg-venv/bin/activate
python demo/app.py
# Opens at http://localhost:7860

Select the attack method in the UI, upload an image, type a hidden message, and click Run.

Run SOTA experiment (CDF accuracy + detectability)

python experiments/cdf_capacity_test.py
# Saves figures to paper/figures/cdf_*.png

Run all Phase 1–3 experiments

python experiments/cdf_capacity_test.py     # Phase 1: CDF steganography
python experiments/pca_test.py              # Phase 2: PCA-guided directions
python experiments/latent_detector_test.py  # Phase 3: Steganalysis detector

Defense: UniversalPatchStegGuard

The demo also shows Joy's defense — UniversalPatchStegGuard from the openclaw package. It uses latent anomaly scoring + spatial smoothing to detect suspicious regions without a secret key.

What it detects

Original PatchSteg (±ε)

Detects the statistical signature of directional perturbations. AUC > 0.90 at ε=5.0.

What evades it

CDF-PatchSteg

Distribution-preserving encoding leaves no detectable anomaly in the latent space. This is the attack-defense narrative of the paper.

Defense gap: No current steganalysis method reliably detects CDF-PatchSteg. This is an open problem and the main contribution of the threat model section.
